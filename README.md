# My-Metapod-Next/UI

![alt text](https://www.icons101.com/icon_png/size_256/id_60004/011_Metapod.png)

A simple UI to display pokemon with login and pokemon features

## Built with:

- Next
- TailwindCSS
- Typescript
- Twin Macro

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
