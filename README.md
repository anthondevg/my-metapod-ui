#My Metapod UI

![alt text](https://images.wikidexcdn.net/mwuploads/wikidex/6/6b/latest/20200307022334/Metapod.png)

A simple UI to display pokemon with login and pokemon features

##Built with:

- Next
- TailwindCSS
- Typescript
- Twin Macro

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
